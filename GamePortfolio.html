Pax Romana

November 2023 - Ongoing

< >

Pax Romana is an ongoing personal project that I'm developing together with friends from Högskolan i Skövde. The game is a paradox inspired grand strategy game with a heavy focus on economy, excluding combat and warfare that is usually present in these types of games.


During this project I've been responsible for alot of the programming work specificly the underlying systems. I've worked together with the other programmer in our group to design and implement different systems like buildings, trade ect. that the player can then interact with.


Since this project might be something we would wanna realese in the future it is not open source. I have however compiled a more detailed portfolio for the project with some samples of code and a presentation that details my work.

Developed with: Unity 2022.3.15f1

Code langues: C#, ShaderLab
